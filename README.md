# iOSCodingChallenges

An iOS application that serves as a collection of coding challenges encountered during technical interviews. It includes problem statements, multiple solution approaches, complexity analysis, and executable Swift implementations to reinforce problem-solving skills and interview readiness.

# Contents

## Spatial Decoder
    a component designed to reconstruct spatial data (such as images, video, or 3D point clouds) from a compressed, non-spatial abstract representation. It upsamples abstract latent vectors or feature maps back into a high-resolution, spatially coherent output.


## Palindrome
    a word, phrase, number, or sequence that reads the same both forward and backward.


## Anagram
    a word or phrase formed by rearranging the letters of another word or phrase. To create a perfect anagram, you must use all the original letters exactly once.
